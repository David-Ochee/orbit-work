## [1.0.0] - 2026-05-16

### Added

- Bounty creation, listing, and management
- Credit-based application system
- Soroban escrow smart contract
- On-chain reputation contract
- Passkey (WebAuthn) wallet integration via `smart-account-kit`
- GitHub OAuth authentication
- GraphQL API with real-time subscriptions (`graphql-ws`)
- PostgreSQL schema and migrations
- Docker Compose local development environment
- GitHub Actions CI/CD workflows
- Schema sync utility for GraphQL schema validation
